# MIDIs folder
Place MIDIs into here using the following structure:

```
midis (You are here)
└──MIDI Name
   └── Version Name.mid
```

The "Version Name" and "MIDI Name" can be anything and does not follow any set standard.

## Example

```
midis (You are here)
├──The Battle
│  └── F1.mid
└──TRR
   ├── F1.mid
   ├── U21.mid
   ├── U20.mid
   ├── U12.mid
   └── U2.mid
